# 📥 Guide de téléchargement des icônes 3D

## 🎯 Objectif
Télécharger 8 icônes 3D pour le composant OrbitingCircles de la page Contact.

## 📍 Emplacement
Les icônes doivent être placées dans : `public/icons/3d/`

## 🔗 Sources recommandées

### Option 1 : Icons8 (GRATUIT - Recommandé)

**Style recommandé** : **3D Fluency** pour une cohérence visuelle

#### Étapes :
1. Visitez [Icons8.com](https://icons8.com)
2. Recherchez chaque icône ci-dessous
3. Sélectionnez le style **"3D Fluency"** ou **"3D Cute"**
4. Téléchargez en **PNG 100px** (gratuit)
5. Renommez selon les noms indiqués

## 🎨 Instructions détaillées pour Icons8

1. **Accédez au site** : https://icons8.com
2. **Recherchez l'icône** (ex: "email 3d")
3. **Filtrez par style** : Sélectionnez "3D Fluency" dans le menu de gauche
4. **Choisissez une icône** qui correspond au thème professionnel
5. **Téléchargez** :
   - Cliquez sur l'icône
   - Sélectionnez la taille : **100 PNG**
   - Cliquez sur **"Download"** (gratuit)
6. **Renommez le fichier** selon les noms ci-dessus
7. **Placez dans** : `public/icons/3d/`
