## 🎓 Comment Utiliser

### 1️⃣ Voir le Site
```bash
# Le serveur est déjà lancé !
# Ouvrez : http://localhost:3000
```

### 2️⃣ Modifier le Contenu
Éditez les fichiers dans `components/sections/` :
- `hero.tsx` → Section d'accueil
- `services.tsx` → Les 4 services
- `expertise.tsx` → Expertise et valeurs
- `about.tsx` → À propos
- `contact.tsx` → Formulaire de contact

### 3️⃣ Modifier les Couleurs
Fichier : `tailwind.config.ts`
```typescript
colors: {
  'odillon-teal': '#1A9B8E',  // Changez ici
  'odillon-lime': '#C4D82E',  // Changez ici
  'odillon-dark': '#0A1F2C',  // Changez ici
}
```

### 4️⃣ Ajouter des Composants
```bash
npx shadcn@latest add dialog --yes
npx shadcn@latest add accordion --yes
npx shadcn@latest add tabs --yes
```

### 5️⃣ Build pour Production
```bash
npm run build
npm start
```
