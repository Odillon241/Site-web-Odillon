## 🎨 Instructions

1. Visitez les liens ci-dessus
2. Choisissez un style 3D cohérent (recommandé : **3D Fluency** ou **3D Cute**)
3. Téléchargez en format PNG, taille 100px
4. Renommez les fichiers selon les noms ci-dessus
5. Placez-les dans ce dossier `public/icons/3d/`
